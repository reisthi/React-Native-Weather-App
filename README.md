# Weather App

Simple weather app to display results from an API call..

## Getting Started

Make sure you have react-native-cli installed

```bash
npm install -g react-native-cli
```

Clone the repo:

```bash
git clone https://github.com/reisthi/WeatherApp.git
```

## Usage

Try 1001 as an input
